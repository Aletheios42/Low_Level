
1. Librería de Funciones Bitwise
printbits - Imprime los bits de un número entero.
abs - Calcula el valor absoluto de un número utilizando operaciones bitwise.
swap - Intercambia dos números usando operaciones bitwise.
count_set_bits - Cuenta el número de bits 1 en un entero.
reverse_bits - Invierte el orden de los bits en un entero.
bit_swap - Intercambia dos bits específicos en un número.
is_power_of_two - Verifica si un número es una potencia de dos.
find_rightmost_set_bit - Encuentra la posición del bit 1 más a la derecha en un entero.
clear_bit_range - Limpia (pone en 0) un rango de bits en un entero.
bitwise_reverse_string - Invierte los caracteres en una cadena utilizando operaciones bitwise.
bitwise_string_compare - Compara dos cadenas de texto utilizando operaciones bitwise para calcular la diferencia entre caracteres.
string_to_uppercase - Convierte toda una cadena a mayúsculas usando operaciones bitwise para manipular los bits del carácter.
clear_nth_bit_in_char - Limpia el enésimo bit en cada carácter de una cadena.
set_nth_bit_in_char - Establece el enésimo bit en cada carácter de una cadena.
string_to_lowercase - Convierte toda una cadena a minúsculas usando operaciones bitwise para manipular los bits del carácter.
toggle_case - Cambia el caso de un carácter en una cadena (mayúscula a minúscula o viceversa) usando operaciones bitwise.

2. Calculadora Aritmética y Congruencia
bitwise_multiply - Multiplica dos enteros utilizando sólo operaciones bitwise.
bitwise_divide - Divide un entero entre otro utilizando sólo operaciones bitwise.
parity_check - Verifica si el número de bits 1 en un entero es par o impar.
3. Algoritmo Base64
Base64 Encoding - Implementa el algoritmo Base64 para codificar datos binarios en una representación textual.
Base64 Decoding - Implementa el algoritmo Base64 para decodificar datos codificados en Base64 de vuelta a su forma binaria original.
4. Algoritmos para Detección de Errores
parity - Implementa la verificación de paridad utilizando operaciones bitwise.
hamming_code - Implementa el código de Hamming para detección y corrección de errores.
checksum - Implementa una suma de verificación básica para detectar errores en datos.
crc - Implementa un algoritmo CRC (Cyclic Redundancy Check) para la verificación de errores en datos.
